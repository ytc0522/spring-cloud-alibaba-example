# Spring Cloud Alibaba Example

## 这个项目是干嘛的？

该项目是用来帮助我个人研究学习Spring Cloud Alibaba 源码的，通过案例代码演示了Spring Cloud Alibaba的核心用法，
通过文档梳理学习框架源码的思路，这样可以更加深入地掌握框架的核心原理。
